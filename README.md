 🗳️ Secure and Transparent Online Voting System

A robust, web-based digital voting platform built with **Java Spring Boot** and **PostgreSQL**. This system focuses on three core pillars: **Security**, **Anonymity**, and **Data Integrity**.

 ✨ Key Features

- **Multi-Factor Authentication (MFA):** Mandatory Email OTP verification during registration to ensure human-only participants.
- **Blockchain-Inspired Integrity:** Implements **Chained Hashing (SHA-256)** where every vote is cryptographically linked to the previous one, making the database tamper-evident.
- **Ballot Secrecy (Anonymity):** A decoupled database architecture that separates voter identity from their specific choices.
- **Digital Receipts:** Voters receive a unique **Receipt Token** to verify that their ballot is recorded in the system without revealing their choice.
- **Real-time Analytics:** A transparent dashboard featuring live result aggregation and dynamic charts using **Chart.js**.
- **Admin Panel:** Centralized management for candidates and system activity logs.

🚀 Getting Started
1. Database Setup
Install PostgreSQL and create a database named Final_Project_Java (or as configured).
The tables will be automatically generated by Hibernate on the first run.

2. Configuration
Update src/main/resources/application.properties:
spring.datasource.url=jdbc:postgresql://localhost:5432/Final_Project_Java
spring.datasource.username=your_postgres_user
spring.datasource.password=your_postgres_password

# For Email OTP (Gmail example)
spring.mail.username=your-email@gmail.com
spring.mail.password=your-app-password

3. Run the Application
Using Gradle:
./gradlew bootRun -x test

Access the app at: http://localhost:8080

🔒 Security Workflow
1.Registration: Email is validated via OTP. Password is saved as a BCrypt hash.
2.Login: Role-based access (ADMIN vs. VOTER).
3.Voting: An Atomic Transaction updates the voter status and inserts an anonymous ballot with a chained hash.
4.Audit: Administrators can view logs, but cannot link individual voters to their votes.
